# 民宿无人入住管理系统

## 项目简介

为民宿提供自助入住解决方案，包含：
- **客人端小程序**：微信小程序，客人自助办理入住
- **管理后台**：Web 端，民宿老板管理入住信息

---

## 技术选型

### 客人端小程序
| 类别 | 方案 |
|------|------|
| 框架 | Taro 3 + React 18 |
| 状态管理 | Zustand |
| 国际化 | i18next |
| 样式 | SCSS + 设计系统 |

### 后端服务
| 类别 | 方案 |
|------|------|
| 运行时 | Node.js + Express |
| 语言 | TypeScript |
| 部署 | 微信云托管 (Serverless) |
| 数据库 | 微信云开发数据库 (NoSQL) |
| 短信 | 阿里云短信 |
| 存储 | 阿里云 OSS |

### 管理后台（二期）
| 类别 | 方案 |
|------|------|
| 框架 | React 18 + Vite |
| UI | MUI |
| 状态 | Zustand |
| 样式 | Tailwind CSS |

## 快速开始

### 小程序端

```bash
cd client
pnpm install
pnpm dev:h5        # H5 开发
pnpm dev:weapp     # 微信小程序开发
```

### 后端服务

```bash
cd server
pnpm install
pnpm dev           # 本地开发 (端口 7001)
pnpm lint          # 代码检查
pnpm lint:fix      # 自动修复
pnpm format        # 格式化代码
```
