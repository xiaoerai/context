## 用户完整流程

```
扫码进入小程序
      ↓
   首页展示
      ↓
点击「办理入住」
      ↓
┌─ 统一弹窗（useCheckinFlow）─────────┐
│  输入手机号 → 短信验证               │
│      ↓                              │
│  选择当日订单                        │
│      ↓ 无入住记录         有入住记录  │
│      ↓                    ↓         │
│  跳转填写入住信息      显示押金支付   │
│  （身份证OCR）             ↓         │
│      ↓ 提交成功          支付成功     │
│      ↓                    ↓         │
│  显示押金支付          显示入住成功   │
│      ↓                              │
│  显示入住成功                        │
└──────────────────────────────────────┘
      ↓
  首页显示门锁密码、WiFi
      ↓
  退房申请 ← ────── checkout.md
      ↓
  押金退还
```
