## 用户完整流程

```
扫码进入小程序
      ↓
   首页展示
      ↓
点击「办理入住」 ──────────────────┐
      ↓                          │
  输入手机号                       │
      ↓                          │
  获取短信验证码  ← ── auth.md     │
      ↓                          │
  选择当日订单                     ├── checkin-flow.md
      ↓                          │
  填写入住信息（身份证OCR）         │
      ↓                          │
  提交入住信息 ───────────────────┘
      ↓
  支付押金 ← ────── deposit.md
      ↓
  入住成功（获取门锁密码、WiFi）
      ↓
  退房申请 ← ────── checkout.md
      ↓
  押金退还
```
