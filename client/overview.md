## 用户流程

> 无需微信授权登录，用户通过"订单手机号 + 短信验证码"验证身份

```
扫码进入小程序
  ↓
首页（展示民宿信息）
  ↓
点击「办理入住」
  ↓
输入订单手机号 → 获取短信验证码 → 验证
  ↓
显示该手机号的当日订单列表
  ↓
选择订单
  ↓
入住信息页 → 上传身份证(OCR自动识别) / 手动填写
  ↓
押金支付页 → 微信支付押金
  ↓
入住成功页 → 查看门锁密码、WiFi
  ↓
退房申请页 → 申请退房，等待确认
```

### 验证流程说明

```
┌──────────────────────────────────────────────────────────────┐
│  为什么用短信验证码？                                         │
│  1. 安全：防止他人偷看订单信息                                │
│  2. 灵活：用户可能用别人手机号订房（帮家人订、公司号等）       │
│  3. 简单：无需微信认证费用（但短信费 ~0.045元/条）            │
└──────────────────────────────────────────────────────────────┘
```

## 页面详情

### 1. 首页 (index)

**功能**：展示民宿信息，提供功能入口

**组件**：
- Hero - 顶部图片 + 民宿名称
- StayCard - 当前入住卡片
- FeatureGrid - 功能入口网格（含"办理入住"入口）
- TabBar - 底部导航

**特点**：深色奢华风格，金色点缀

---

### 2. 办理入住弹窗 (CheckInModal)

**功能**：验证身份 + 选择订单（两步合一）

**触发**：点击首页 FeatureGrid 中的「办理入住」入口

**Step 1 - 手机号验证**：
- 手机号输入框（11位）
- 验证码输入框（4-6位）
- 获取验证码按钮（60秒倒计时）

**Step 2 - 选择订单**：
- 验证通过后显示当日订单列表
- 订单卡片：房间号、入住日期、退房日期
- 点击订单 → 关闭弹窗 → 跳转入住信息页

**流程**：
```
┌─────────────────────────────────┐
│  Step 1: 验证手机号              │
│  ┌─────────────────────────┐    │
│  │ 手机号: [___________]   │    │
│  │ 验证码: [____] [获取]   │    │
│  │                         │    │
│  │      [下一步]           │    │
│  └─────────────────────────┘    │
└─────────────────────────────────┘
              ↓ 验证通过
┌─────────────────────────────────┐
│  Step 2: 选择订单                │
│  ┌─────────────────────────┐    │
│  │ 🏠 301室                │    │
│  │ 3月9日 - 3月11日        │    │
│  └─────────────────────────┘    │
│  ┌─────────────────────────┐    │
│  │ 🏠 502室                │    │
│  │ 3月9日 - 3月10日        │    │
│  └─────────────────────────┘    │
└─────────────────────────────────┘
              ↓ 点击选择
       跳转入住信息页 (checkin)
```

**特点**：
- 验证码有效期5分钟
- 无订单时提示"该手机号今日无待办理订单"
- 可返回 Step 1 更换手机号

---

### 3. 入住信息页 (checkin)

**功能**：填写入住登记信息，上传证件

**表单字段**：
- 姓名（必填）
- 身份证号（必填，18位校验）
- 手机号（必填，11位校验）

**上传内容**：
- 身份证正面（可选，OCR自动识别填充表单）

**验证规则**：
- 身份证号：18位有效格式
- 手机号：11位数字
- 使用 zod 统一验证

---

### 4. 押金支付页 (deposit)

**功能**：显示押金信息，调起微信支付

**展示内容**：
- 房间信息
- 入住/退房日期
- 押金金额

**流程**：
```
显示押金 → 点击支付 → 微信支付 → 成功跳转
```

---

### 5. 入住成功页 (success)

**功能**：展示入住信息和门锁密码

**展示内容**：
- 房间号
- 门锁密码（大字醒目）
- WiFi 名称 + 密码
- 入住/退房时间

**操作**：
- 联系房东
- 申请退房

---

### 6. 退房申请页 (checkout)

**功能**：提交退房申请

**展示内容**：
- 入住信息
- 退房状态（待检查/已退房）

**流程**：
```
申请退房 → 等待房东确认 → 退还押金
```
